# 🗺️ 事件与大地图调试文档

本文件面向开发者，记录所有事件节点的选项与对应跳转结果。游戏中不会向玩家显示这些结果。

---

## 📍 节点总览

当前世界由3个地图组成，共15个节点：

| 地图        | 节点数 | 节点列表                                          |
|:---------- |:----:|:----------------------------------------------- |
| 永冻雪地     | 5    | snow_start, snow_1, snow_shop, snow_2, snow_boss   |
| 风暴海岸     | 5    | coast_start, coast_1, coast_shop, coast_2, coast_boss |
| 瘴气丛林     | 5    | jungle_start, jungle_1, jungle_shop, jungle_2, jungle_boss |

---

## 🌅 起始事件

### start - 起始之地

**描述**：你已经来到了这片大地，你需要做的只是寻宝，关于一切都不需要在意了。

| 玩家可见选项                | 对应结果                                  |
|:----------------------- |:--------------------------------------- |
| 睁开眼睛发现身处大陆        | 随机进入 snow_start / coast_start / jungle_start |
| 转过身感到一片寒意          | 进入 snow_start（雪地入口）                  |
| 抬起头发现植被茂密          | 进入 jungle_start（丛林边缘）                |

## ❄️ 永冻雪地

### snow_start - 雪地入口

**描述**：寒风裹挟着碎雪扑面而来，远处有模糊的脚印延伸向白茫茫的深处。

| 玩家可见选项      | 对应结果                  |
|:-------------- |:----------------------- |
| 沿着脚印前进      | 进入 snow_1（冰封峡谷，战斗） |
| 向着炊烟方向走    | 进入 snow_shop（雪原补给站）  |

### snow_1 - 冰封峡谷

**类型**：combat
**描述**：冰层下传来不祥的震动，一只雪原怪物从裂隙中爬出！
**胜利后**：自动进入 snow_2

### snow_shop - 雪原补给站

**类型**：shop
**描述**：旅行商人支起了帐篷，锅里煮着热汤。要补充些什么吗？
**离开商店后**：自动进入 snow_2

### snow_2 - 霜风高地

**描述**：高地的风雪中矗立着三座路标，分别指向不同的方向。

| 玩家可见选项      | 对应结果                    |
|:-------------- |:------------------------- |
| 继续深入雪地      | 进入 snow_boss（雪顶巨像，Boss） |
| 向东穿越海峡      | 进入 coast_2（海崖哨塔）       |
| 向南进入针叶林    | 进入 jungle_1（毒瘴沼泽，战斗）  |

### snow_boss - 雪顶巨像

**类型**：boss
**描述**：暴风雪的中心，一尊由寒冰与骸骨堆砌的巨像缓缓站起。
**击败后选项**：

| 玩家可见选项          | 对应结果              |
|:------------------ |:------------------- |
| 下山前往海岸          | 进入 coast_start     |
| 穿越雪林前往丛林       | 进入 jungle_start    |
| 返回世界起点          | 进入 start           |

## 🌊 风暴海岸

### coast_start - 海岸沙滩

**描述**：腥咸的海风拍打着你的脸颊，沙滩上散落着破碎的贝壳与船板。

| 玩家可见选项      | 对应结果                    |
|:-------------- |:------------------------- |
| 沿着海岸线走      | 进入 coast_1（礁石滩，战斗）   |
| 检查搁浅的船只    | 进入 coast_shop（渔人码头）    |

### coast_1 - 礁石滩

**类型**：combat
**描述**：潮水退去，隐藏在礁石后的生物向你看过来。
**胜利后**：自动进入 coast_2

### coast_shop - 渔人码头

**类型**：shop
**描述**：老渔夫正在修补渔网，他愿意用寻来的宝物与你交易。
**离开商店后**：自动进入 coast_2

### coast_2 - 海崖哨塔

**描述**：破败的哨塔上挂着一盏忽明忽暗的灯，塔下有三条路。

| 玩家可见选项      | 对应结果                     |
|:-------------- |:-------------------------- |
| 潜入海底洞穴      | 进入 coast_boss（深渊海妖，Boss） |
| 向北攀登雪山      | 进入 snow_1（冰封峡谷，战斗）      |
| 向内陆行进        | 进入 jungle_2（古树祭坛）          |

### coast_boss - 深渊海妖

**类型**：boss
**描述**：海水骤然变黑，巨大的触手从漩涡中伸出。
**击败后选项**：

| 玩家可见选项      | 对应结果           |
|:-------------- |:---------------- |
| 前往雪地          | 进入 snow_start  |
| 前往丛林          | 进入 jungle_start |
| 返回世界起点      | 进入 start        |

## 🌿 瘴气丛林

### jungle_start - 丛林边缘

**描述**：湿热的气流夹杂着腐叶味，藤蔓像触手一样垂落在小径两侧。

| 玩家可见选项      | 对应结果                      |
|:-------------- |:--------------------------- |
| 踩着小径前进      | 进入 jungle_1（毒瘴沼泽，战斗）  |
| 寻找土著营地      | 进入 jungle_shop（草药商人）      |

### jungle_1 - 毒瘴沼泽

**类型**：combat
**描述**：脚下的淤泥冒出气泡，毒虫从腐木中涌出！
**胜利后**：自动进入 jungle_2

### jungle_shop - 草药商人

**类型**：shop
**描述**：一位蒙面商人正在研磨草药，她的货架上摆满奇异的瓶罐。
**离开商店后**：自动进入 jungle_2

### jungle_2 - 古树祭坛

**描述**：一棵参天古树的树干上刻满了符号，树下有三条隐蔽的小道。

| 玩家可见选项      | 对应结果                       |

### jungle_boss - 丛林主宰

**类型**：boss
**描述**：古树的根系开始蠕动，一头被藤蔓缠绕的巨兽从地下钻出。

| 前往雪地          | 进入 snow_start  |
| 前往海岸          | 进入 coast_start |
| 返回世界起点      | 进入 start        |

## ⚙️ 调试说明

### 节点数据结构

```javascript
{
    id: '节点唯一标识',
    name: '节点显示名称',
    type: 'start|event|combat|shop|boss',
    icon: 'emoji图标',
    desc: '玩家看到的描述文本',
    options: [ // event/start/boss 类型使用
        { text: '玩家可见选项', result: { type: 'goto|random', target: '目标节点', targets: ['随机目标1','随机目标2'] } }
    ],
    next: '自动跳转的下一个节点', // combat/shop 类型使用
    mobBias: '怪物偏好', // combat 类型可选
    bossId: 'boss标识' // boss 类型可选
}
```

### 通关判定

击败 `bossNodes` 数组中的所有Boss节点（snow_boss, coast_boss, jungle_boss）即可通关。

### 难度公式

敌人强度 = 基础属性 × (1 + visitedNodes.length × 0.15)

*Boss额外倍率*：生命 × 1.8，攻击 × 1.4
