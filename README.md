# 🗡️ 12俊俊寻宝记 x 联动扩展版

一个基于纯前端技术（HTML/CSS/JavaScript）开发的轻量级 roguelike 寻宝冒险游戏。

## 🎮 在线体验

直接在浏览器中打开 `index.html` 即可开始游戏，无需任何构建工具或服务器环境。

## ✨ 核心玩法

### 角色选择
- **soso5网格** 🕺 —— 每3回合触发随机舞蹈（闪避/暴击/回血/综合）
- **快餐侠洲歌** 🍔 —— 每回合随机扣自己或对方10%血量，致死自残可保留1血并反杀
- **抖抖鸡** 🐔 —— 每回合额外闪避判定，每次闪避成功攻击+1

### 世界探索
游戏包含三大区域、15个节点的大地图：
- ❄️ **永冻雪地** —— 网格地图探索，遭遇冰原专属怪物与Boss"菇菇祭祀"
- 🌊 **风暴海岸** —— 礁石滩战斗、渔人码头交易
- 🌿 **瘴气丛林** —— 毒瘴沼泽、古树祭坛与丛林主宰

### 战斗系统
- 自动回合制战斗，按速度决定出手顺序
- 暴击、闪避、吸血、冰冻、冰盾、威慑等多层机制
- 终结技系统：每个角色拥有独特的终极技能
- 镇镇之力隐藏觉醒系统：战斗中满足特定条件可解锁隐藏能力

### 道具与成长
- 战斗胜利后选择战利品（属性强化、回血、唯一神器）
- 商店购买装备（陨铁剑、疾风鞋、吸血镰刀等）
- 背包系统记录已获得的唯一道具
- 官方图鉴系统查阅所有角色、怪物与道具信息

## 🗂️ 项目结构

```
.
├── index.html          # 主页面与UI结构
├── style.css           # 全局样式与战斗/地图/模态框样式
├── data.js             # 角色、怪物、道具、镇镇之力数据定义
├── events.js           # 大地图节点、冰原事件池、网格地图生成
├── game.js             # 核心游戏逻辑（战斗、探索、状态管理）
├── events.md           # 事件节点调试文档（开发者参考）
└── 🗡️ 12俊俊寻宝记 - 官方开发者图鉴v0.1.md
```

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/flierpig/12-finding.git

# 进入目录
 cd 12-finding

# 直接用浏览器打开
open index.html
```

或使用任意本地静态服务器：

```bash
npx serve .
# 或
python -m http.server 8080
```

## 🛠️ 技术栈

- **HTML5** —— 语义化页面结构
- **CSS3** —— 渐变背景、动画、网格布局、响应式模态框
- **原生 JavaScript** —— 无框架依赖，模块化数据与逻辑分离
